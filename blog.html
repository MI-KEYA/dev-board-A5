Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using `getElementById()` for an element by ID, or `querySelector()` to select elements using CSS selectors. Other methods include `getElementsByClassName()` and `getElementsByTagName()`.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets or sets the HTML content (including tags) inside an element. innerText: Gets or sets the visible text content, excluding hidden elements or styles. textContent: Gets or sets the text content, including hidden elements, but ignores HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of attaching an event listener to each individual child element, you attach it to a parent element and use event bubbling to handle events for child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event triggered on an element first runs on that element, then propagates (or "bubbles up") to its parent elements, and continues up the DOM tree until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element with `document.createElement('div')`, add it with `parentElement.appendChild(newElement)`, and remove it with `newElement.remove()` or `parentElement.removeChild(newElement)`.
